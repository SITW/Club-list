# Club-list 

using Angular.js + Yeoman

## Required

- npm 
- nodejs

## Prepare

install bower packages

```
  bower install
```

install npm packages

```
  npm install
```

## Usage 

Run on local
- grunt server

## Deploy

    $ grunt build

add go to root file

    $ git subtree push --prefix dist origin gh-pages
